# Volume Commands

These commands in the PPT are used to manage Docker volumes.

---

## Create a Volume

```bash
docker volume create myvolume
```

Creates a persistent storage volume.

---

## List Volumes

```bash
docker volume ls
```

Shows all Docker volumes.

---

## Inspect a Volume

```bash
docker volume inspect myvolume
```

Displays detailed information about a volume.

---

## Remove a Volume

```bash
docker volume rm myvolume
```

Deletes the specified volume.
