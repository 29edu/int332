# Network Commands

These commands in the PPT are used to manage Docker networks.

---

## List Networks

```bash
docker network ls
```

Shows available Docker networks.

---

## Create a Network

```bash
docker network create mynetwork
```

Creates a custom Docker network.

---

## Inspect a Network

```bash
docker network inspect mynetwork
```

Displays detailed network configuration.

---

## Remove a Network

```bash
docker network rm mynetwork
```

Deletes the specified Docker network.
